# Agent Studio kinship

Quality Studio intentionally shares Agent Studio's compact desktop-workbench vocabulary: layered app/editor/surface backgrounds, 5–8 px radii, thin low-contrast borders, system UI type beside monospace technical content, inset selection rails, restrained accent color, and small circular status signals. The source tokens use the `--studio-*` namespace so useful additions can move back without translation.

Quality-specific semantics extend the family language rather than replacing it: green means fresh/current, amber means stale/attention, and grey means missing/not reviewed. Review lanes reuse the compact elevated-card rhythm while the tree and editor adopt the same dense pane headers and tab treatment. Both themes use the identical spatial and semantic token structure.
